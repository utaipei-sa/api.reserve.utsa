# reserve.utsa
A platform to reserve the spaces and items managed by UTSA.

Develop documentation: [Wiki](https://github.com/utaipei-sa/reserve.utsa/wiki)

How to run:  

**back end**  
1. `cd server`  
2. `npm install`  
3. `nodemon` (if you don't need auto reload, you can use `npm start`)  

**fornt end**  
1. `cd appfront`  
2. `npm install`  
3. `npm run dev`
